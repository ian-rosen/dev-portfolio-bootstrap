## Overview
This repository contains a simple, mobile-first portfolio template (`index.html`) and accompanying styles (`assets/css/style.css`). It provides a clean starting point to customize and deploy a personal developer portfolio.

## Features
- Minimal, responsive layout with sections for an about summary, project list, and contact information.
- Easy to customize design tokens and styles in `assets/css/style.css`.
- Static, no build step — ready to deploy to GitHub Pages or any static host.

## Technologies
- **Languages:** HTML, CSS
- **Tools:** Any static host (GitHub Pages, Netlify, Vercel)

## How to Use

### Local Setup
1. Clone the repo:

```bash
git clone https://github.com/your-username/dev-portfolio-bootstrap.git
cd dev-portfolio-bootstrap
```

2. Open the site in your browser by opening `index.html`:

Mac / Linux:
```bash
open index.html
```

Windows (PowerShell):
```powershell
start index.html
```

### Deploy
- For GitHub Pages: push the repo to GitHub and enable Pages from the repository settings (branch: `main` or `gh-pages`).

## Project Structure
- `index.html` — main HTML file
- `assets/css/style.css` — core stylesheet
- `assets/images/` — place project screenshots and avatars here

## What I Learned
